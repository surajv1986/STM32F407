This project seeks to read pressure, temperature and compute equivalent sea level and baseline pressure using a Bosch BMP180 I2C sensor
Developed using HAL Library and Keil IDE
Hardware Required
1)STM32F407 Microcontroller
2)Bosch BMP180 sensor


Hardware Connections
STM32F407      BMP180
PB6            SCL
PB7			   SDA
3.3V           Supply Pin
GND            GND
